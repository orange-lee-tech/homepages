# 研究定位

我走的是 **「可復現實驗 + 工程落地」** 的研究路徑：用科學計算與機器學習結合，優先解決真實物理系統裡的 **穩定性、泛化與可解釋性** 問題。

# 目前主線（在做什麼）

- 方向：物理約束增強神經算子（Neural Operator / Physics-constraint / Projection）
- 目標：把方法做成「能復現、能對比、能解釋」的證據鏈（實驗 + 消融 + 失敗模式）
- 輸出：論文級實驗結果（圖表/指標/程式碼結構）+ 可展示的 demo

# 我能帶來的價值（合作視角）

- 把想法做成可交付：有數據、對比、有結論、有復現入口
- 把工程做成證據：指標口徑清晰，錯誤能定位，迭代有節奏
- 把表達做成資產：寫作/報告面向評審與讀者，不堆疊術語
